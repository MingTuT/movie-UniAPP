# 电影推荐

## 项目简介

该项目模仿《猫眼》与《豆瓣》的页面设计，是一款推荐优质电影的应用。

搭建该项目使用的是Uni-App框架，该框架集成了许多API，十分简便容易上手。

后端使用的技术是：Node.js 中的 Express 框架 + MongoDB 数据库 。
